# Maze Wanderer

A simple first person raycasting demo, written in [Rust](https://rust-lang.org).


## Controls

`X` - Start

`Z` - Toggle betweeen modes

### First Person Mode

`ᐊ`, `ᐃ`, `ᐁ`, `ᐅ` - Move selection

`X` - Toggle cell

### Map Editor Mode

`ᐊ`, `ᐅ` - Look

`ᐃ`, `ᐁ` - Move

`X` - Sprint
